# 196. Building with Webpack or ESBuild 🏗️

## Section 15: Creating & Running TypeScript Projects with Vite

To solve full project build needs, common tools:
- **Webpack**
- **esbuild**

## What they help with

- handling multiple file types
- optimizing/bundling assets
- preparing deploy-ready output

## Reality check

You don’t always need to configure these tools from scratch manually.  
Often, projects already come pre-configured.

## Para sa non-developer 👇

Parang logistics system na hindi lang nagta-translate text, kundi nagpa-package ng buong shipment.

## Recap ✅

- Webpack/esbuild are powerful build engines for complete web app pipelines
