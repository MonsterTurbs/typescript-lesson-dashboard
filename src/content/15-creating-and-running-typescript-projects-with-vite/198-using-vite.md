# 198. Using Vite 🧪

## Section 15: Creating & Running TypeScript Projects with Vite

Typical Vite setup flow:

```bash
npm create vite@latest
cd your-project
npm install
npm run dev
npm run build
```

## What happens

- dev mode: live server + quick updates
- build mode: production-ready optimized output in `dist`

## Para sa non-developer 👇

Parang may “draft mode” at “publish mode” ka in one tooling workflow.

## Recap ✅

- Vite makes local development and production build workflow straightforward
