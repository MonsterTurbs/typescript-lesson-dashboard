# 197. Introducing Vite ⚡

## Section 15: Creating & Running TypeScript Projects with Vite

Vite is a modern tool that gives you:
- fast dev server
- TypeScript-ready setup
- build pipeline for real frontend projects

## Why devs like Vite

- fast startup and rebuilds
- simple project scaffolding
- smooth integration with modern frameworks or vanilla TS setups

## Para sa non-developer 👇

Parang ready-made construction kit: hindi ka na magsisimula sa bare metal.

## Recap 🚀

- Vite provides practical, modern default tooling for TS web development
