## Section 15: Creating & Running TypeScript Projects with Vite

So far, marami na tayong TS concepts na natutunan.  
Ngayon, focus tayo sa mas realistic project setup gamit build tools like **Vite**. 🚀

## Why this section matters

- real projects are multi-file and mixed assets (`.ts`, `.html`, `.css`, etc.)
- `tsc` alone is often not enough for full web project builds
- Vite helps you run and build faster with modern tooling
