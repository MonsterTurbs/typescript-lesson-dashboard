# 200. Making Sense of .d.ts Files 📄

## Section 15: Creating & Running TypeScript Projects with Vite

`.d.ts` files are **declaration files**:
- type info only
- no runtime executable logic

## Purpose

- provide TypeScript type definitions
- improve IDE support/autocomplete/type checking
- commonly used by libraries

## Para sa non-developer 👇

Parang specification sheet lang siya, hindi actual machine code.

## Recap ✅

- `.d.ts` files describe types, not behavior execution
