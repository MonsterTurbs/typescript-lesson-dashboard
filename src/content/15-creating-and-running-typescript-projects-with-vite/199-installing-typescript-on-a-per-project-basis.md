# 199. Installing TypeScript on a Per-Project Basis 📌

## Section 15: Creating & Running TypeScript Projects with Vite

Best practice: install TypeScript per project, not only globally.

## Why this helps

- project works even if global TS isn’t installed
- you can pin TS version per project
- safer compatibility for legacy/older codebases

## Para sa non-developer 👇

Parang bawat branch office may sariling approved toolkit version para stable operations.

## Recap 🎯

- Per-project TS dependency gives consistency and version control safety
