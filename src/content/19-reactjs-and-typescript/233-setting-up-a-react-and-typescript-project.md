# 233. Setting Up a React + TypeScript Project 🛠️

Pinakamadali: gumamit ng Vite starter.

```bash
npm create vite@latest
# choose: React -> TypeScript
```

Pagkatapos:

```bash
npm install
npm run dev
npm run build
```

## Why Vite?

- mabilis dev server
- built-in TS support
- smooth production build flow

## Para sa non-developer 👇

Parang ready-made kitchen setup: kumpleto na ang basic tools, magluluto ka na lang ng app.
