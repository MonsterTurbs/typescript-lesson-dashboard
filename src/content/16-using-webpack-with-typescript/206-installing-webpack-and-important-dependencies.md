# 206. Installing Webpack & Important Dependencies 🧰

## Section 16: Using Webpack with TypeScript

Typical install command:

```bash
npm install --save-dev webpack webpack-cli webpack-dev-server ts-loader typescript
```

## What each package does

- `webpack`: core bundler
- `webpack-cli`: run webpack commands
- `webpack-dev-server`: local dev server with rebuild flow
- `ts-loader`: tells webpack how to handle `.ts`
- `typescript`: compiler dependency pinned per project

## Para sa non-developer 👇

Parang bumili ng buong toolkit set para gumana ang assembly line.

## Recap ✅

- These dev dependencies form the base TS + Webpack toolchain
