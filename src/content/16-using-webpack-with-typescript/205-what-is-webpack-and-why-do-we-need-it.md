# 205. What is Webpack & Why do we need it? 📦

## Section 16: Using Webpack with TypeScript

Problem sa multi-file ES module apps: maraming HTTP requests.

## Why this hurts

Kahit maliit bawat file, bawat request may setup overhead.  
Mas maraming requests = potential latency.

## What Webpack does

- bundles files together
- reduces request count
- can optimize/minify output
- supports broader build pipeline customization

## Para sa non-developer 👇

Parang imbes 20 maliit na deliveries, pinagsama sa iilang efficient shipments.

## Recap 🎯

- Webpack solves packaging/performance orchestration for bigger frontend codebases
