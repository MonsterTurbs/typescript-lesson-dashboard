# 247. Node & TypeScript: Options & Alternatives 🔀

May iba’t ibang paraan para patakbuhin ang TS sa server world.

## Common options

- `tsc` compile to JS, then run with `node`
- native TS support sa modern Node (with limits)
- alternatives: Deno / Bun (built-in TS support)

## Sample classic flow

```bash
npm run compile
node dist/app.js
```

## Para sa non-developer 👇

Parang iba-ibang sasakyan papunta sa same destination: parehong app, iba lang ang runtime/tooling path.
