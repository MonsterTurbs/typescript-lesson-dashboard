# 256. Exploring Alternative Runtimes Like Bun 🐇

Bukod sa Node, may alternatives gaya ng Bun at Deno.

## Why explore them?

- built-in TypeScript support
- potential speed/runtime features
- different developer experience

Example with Bun:

```bash
bun run app.ts
```

## Reminder

Node pa rin ang pinaka-common sa maraming projects, pero good to know ang options.

## Para sa non-developer 👇

Parang pagpili ng ibang engine na puwedeng magpatakbo ng parehong app, depende sa needs mo.
