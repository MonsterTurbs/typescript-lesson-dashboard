## Section 7: Advanced Types

Ready na for deeper TypeScript! 🔥  
Sa section na ito, papasok tayo sa mas advanced patterns para mas robust at scalable ang code.

## Ano ang coverage?

- intersection types
- type guards
- discriminated unions
- `instanceof` checks
- function overloads
- index types
- `as const`
- `Record`
- `satisfies`

## Bakit important ito?

Kapag lumalaki ang app:
- mas maraming possible data shapes
- mas kailangan ng strict type safety
- mas kailangan ng malinaw na narrowing rules

## Para sa non-developer 👇

Parang level-up ito ng quality control:
- hindi lang "gumagana"
- kundi "safe, clear, at predictable" kahit complex na ang system
