# 183. Writing Module Code - Your Options 🧭

## Section 14: Modules & Namespaces

May dalawang main approaches sa TypeScript when splitting code:

1. Namespaces + TS-style references/bundling  
2. ES Modules (`import` / `export`) na standard JavaScript way

## Quick comparison

- `namespace`: older TS-centric workflow
- `ES modules`: modern standard, mas recommended sa most projects

## Para sa non-developer 👇

Parang may old filing system at may modern cloud-linked system.  
Parehong puwede, pero usually mas future-friendly ang modern.

## Recap 🎯

- You should know both, but ES Modules usually win for maintainability
