# 185. Organizing Files & Folders 🗂️

## Section 14: Modules & Namespaces

As app grows, folder structure matters a lot.

## Suggested structure

- `components/`
- `models/`
- `state/`
- `util/`
- `decorators/`

## Why this helps

- easier file discovery
- cleaner mental map
- less chaos in large codebases

## Para sa non-developer 👇

Parang office filing cabinet:
- hiwalay ang contracts, reports, finance docs
- mas mabilis hanapin, mas konting mali

## Recap 🚀

- Good folder organization improves speed and maintainability
