## Bakit exciting ang TypeScript? 😄

TypeScript builds on top of JavaScript.
Ibig sabihin, hindi ka mag-uumpisa from zero.

Daladala mo pa rin ang JS knowledge mo,
pero mas gaganda ang workflow mo dahil mas guided ang coding experience.

Sa madaling salita:
- same familiar JavaScript foundation
- better structure
- early error detection
- mas reliable na project output ✅
