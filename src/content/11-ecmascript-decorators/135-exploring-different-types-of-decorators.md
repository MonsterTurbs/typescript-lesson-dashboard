# 135. Exploring Different Types of Decorators 🧩

## Section 11: ECMAScript Decorators

Important: decorators are object-oriented feature scope.

## Saan puwedeng i-attach?

- classes
- class methods
- class fields
- getters/setters

## Saan hindi puwede?

- regular standalone functions
- plain variables outside class members

## Why this matters

Para clear agad kung anong targets supported bago ka mag-design ng decorator logic.

## Para sa non-developer 👇

Parang accessory system na compatible lang sa specific device parts.  
Hindi lahat ng surface puwedeng lagyan.
