## Section 8: Generic Types

Welcome sa Section 8! 🎉  
Dito natin aaralin ang **Generics**: isang concept na mukhang intimidating sa una, pero super useful kapag na-gets mo na.

## Ano ang matututunan mo dito?

- ano ang generic types
- bakit sila nag-eexist
- paano gumawa ng sariling generic types
- paano gamitin generics sa functions, classes, at interfaces

## Bakit important ito?

Generics help you build:
- flexible types
- reusable utilities
- strong type safety without hardcoding everything

## Para sa non-developer 👇

Parang adjustable container ito:
- iisang lalagyan
- puwedeng i-configure para sa iba-ibang laman
- pero controlled pa rin ang rules
