## Section 13: Practice Time! Let's build a Drag & Drop Project

Ihahanda natin ang starter files:
- `index.html` with templates
- `app.css` for styles
- `app.ts` for TypeScript logic

## Why templates?

`<template>` lets us keep HTML structures na hindi agad nari-render, then i-insert via TS.

```html
<template id="project-input">...</template>
<template id="single-project">...</template>
<template id="project-list">...</template>
```
