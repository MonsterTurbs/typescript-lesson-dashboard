## Ano ang Linked List?

Linked list is a data structure made of **nodes**.

Each node usually has:
- `value` (actual data)
- `next` (reference sa susunod na node)

## Visual idea

`[10] -> [5] -> [-3] -> null`

Bawat box = node.  
Arrow = link sa next node.

## Linked List vs Array (quick idea)

- Array: direct index access (`arr[2]`)
- Linked List: step-by-step traversal node by node

May tradeoffs pareho.  
Sa some operations, linked list can be more efficient.
