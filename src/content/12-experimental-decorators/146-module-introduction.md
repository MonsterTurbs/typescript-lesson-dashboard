# 146. Module Introduction 🧪

## Section 12: Experimental Decorators

Welcome sa next decorators deep-dive! 🚀  
Ngayon naman, focus tayo sa **experimental/legacy decorators** ng TypeScript.

## Bakit ito important?

- ito pa rin ang madalas makita sa existing TS projects
- maraming libraries/frameworks ang naka-base dito
- practical para sa maintenance ng older codebases

## Quick setup reminder

- `tsconfig.json`: set `experimentalDecorators: true`
- optional: `npm install` + `tsc -w` for watch mode

## Para sa non-developer 👇

Parang natutunan mo na ang bagong official format,  
ngayon aaralin mo ang “older widely-used format” na ginagamit pa rin sa maraming apps.
