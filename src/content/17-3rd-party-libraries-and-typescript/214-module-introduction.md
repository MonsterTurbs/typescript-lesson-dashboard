## Section 17: 3rd Party Libraries & TypeScript

Sa real projects, hindi lang pure TypeScript ang gamit.  
Madalas, may mga third-party libraries din na kasama. 🚀

## Focus ng section na ito

- paggamit ng JavaScript libraries sa TypeScript projects
- pag-intindi sa `.d.ts` files
- TypeScript-first libraries (e.g. Zod-style workflow)

## Bakit important ito?

Kasi dito nagsisimula ang tunay na integration work sa production apps.

## Para sa non-developer 👇

Parang teamwork ito ng internal app code + external tools.  
Kailangan compatible silang mag-usap para iwas bug.
