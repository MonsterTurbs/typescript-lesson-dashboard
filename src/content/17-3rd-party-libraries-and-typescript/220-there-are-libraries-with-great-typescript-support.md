# 220. There Are Libraries with Great TypeScript Support 🌟

Good news: maraming libraries ngayon na TypeScript-friendly na out of the box.

Examples:
- `zod`
- `prisma`
- ibang TS-first tools

## Difference vs pure JS libs

- Pure JS lib: madalas kailangan pa ng `@types/...`
- TS-ready lib: may built-in typings na agad

```bash
npm install zod
```

## Para sa non-developer 👇

Parang appliance na may kasamang kumpletong manual at safety labels na agad.
Mas mabilis i-setup, mas konti guesswork.
